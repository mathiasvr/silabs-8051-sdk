Program Description:
--------------------------------------------------------------------------------
This example code demonstrates the power consumption and use of different 
CPU work mode with external oscillator. 

Resources Used:
--------------------------------------------------------------------------------
WDT0 (Watchdog Timer - Disabled)

Product Family:
--------------------------------------------------------------------------------
C8051F91x/90x

Datasheet Version:
--------------------------------------------------------------------------------
Rev 1.2 

Specs Demonstrated:
--------------------------------------------------------------------------------
Table 4.2 Digital Core (SYSCLK) - Normal Mode
   - Frequency = 1 MHz (External osciallator)
      + Min: --
      + Typ: 305 uA
      + Max: --   

Table 4.2 Digital Core (SYSCLK) - Idle Mode   
   - Frequency = 1 MHz (External osciallator)
      + Min: --
      + Typ: 185 uA
      + Max: --

--------------------------------------------------------------------------------
