Program Description:
--------------------------------------------------------------------------------
This application will communicate with a PC across the USB interface.
The device will appear to be a mouse, and will manipulate the cursor
on screen.


How to Use:
--------------------------------------------------------------------------------
 1) Connect the ToolStick342DC to a ToolStick Base Adapter.
 2) Compile and download code to the ToolStick Daughter Card by selecting
    Run -> Debug from the menus, clicking the Debug button in the quick menu,
    or pressing F11.
 3) Run the code by selecting Run -> Resume from the menus, clicking the
    Resume button in the quick menu, or pressing F8.
 4) Connect the provided Micro-B USB cable from a PC to J2 on the ToolStick 
	Daughter Card.
 5) The device will appear as an HID mouse on the PC. Turn the potentiometer
	to move the cursor left or right.
