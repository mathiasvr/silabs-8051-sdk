EFM8 USB Type-C High Power Charger Reference Design
--------------------------------------------------------------------------------
 
Description:

This reference design Implements a USB Type-C High Power Charger.

The charger can provide up to 60W of power.

This reference design is meant to be run on the USB Type-C 60 W Charger Board
(BRD5204A).
