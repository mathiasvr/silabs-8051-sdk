Program Description:
--------------------------------------------------------------------------------
This example code demonstrates the power consumption and use of different 
processor work mode with external oscillator. 

Resources Used:
--------------------------------------------------------------------------------
WDT0 (Watchdog Timer - Disabled)

Product Family:
--------------------------------------------------------------------------------
C8051F93x/92x

Datasheet Version:
--------------------------------------------------------------------------------
Rev 1.3 

Specs Demonstrated:
--------------------------------------------------------------------------------
Table 4.2 Digital Core (SYSCLK) - Normal Mode
   - Frequency = 1 MHz (External oscillator)
      + Min: --
      + Typ: 365 uA
      + Max: --   

Table 4.2 Digital Core (SYSCLK) - Idle Mode   
   - Frequency = 1 MHz (External oscillator)
      + Min: --
      + Typ: 235 uA
      + Max: --

  
How to Use:
--------------------------------------------------------------------------------
Refer to F93x_Clock_External.c for detailed directions on using this example.
