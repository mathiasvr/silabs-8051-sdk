Program Description:
--------------------------------------------------------------------------------
This program demonstrates the use of the Device ID to create device specific
programs that allow the same firmware image to run on MCUs with different 
memory sizes and peripherals, and dynamically change functionality to suit the 
capabilities of that MCU.
