Program Description:
--------------------------------------------------------------------------------
This code is used to show the power consumption of the Internal Precision
Oscillator. 

Resources Used:
--------------------------------------------------------------------------------
WDT0 (Watchdog Timer - Disabled)
P0.0 (Switch Precision Oscillator ON/OFF)

Datasheet Version:
--------------------------------------------------------------------------------
Rev 0.5

Specs Demonstrated:
--------------------------------------------------------------------------------
Table 4.9 Internal Precision Oscillator
    - Oscillator Supply Current
       + Min: --
       + Typ: 300 uA
       + Max: --

How to Use:
--------------------------------------------------------------------------------
Refer to F96x_PR_Oscillator_P.c for detailed directions on using this example.
